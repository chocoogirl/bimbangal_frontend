# Bimbangal Frontend

### Pre-requisites

* npm
* React
* Redux
* React router dom
* Axios

### Important

Run the backend first and change the API BASE_URL in config in frontend before running

### Install

* Clone the repo
* Run `npm install`
* Run `npm start`
 
    ```
    $ git clone CLONE_URL
    $ cd CLONED_FOLDER
    $ npm install
    $ npm start
    $ Website is served at LOCALHOST
    ```
